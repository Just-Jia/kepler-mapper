kmapper.GraphNerve¶

class kmapper.GraphNerve(min_intersection=1)[source]¶

Creates the 1-skeleton of the Mapper complex.

Parameters:min_intersection (int, default is 1) – Minimum intersection considered when computing the nerve. An edge will be created only when the intersection between two nodes is greater than or equal to min_intersection
